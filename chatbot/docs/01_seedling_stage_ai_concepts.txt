1단계: 새싹 단계 – AI의 개념 잡기
==================================

이 단계에서는 AI의 기본 개념과 전체적인 흐름을 이해합니다.

1.1 AI란 무엇인가
----------------
인공지능(AI)은 컴퓨터가 인간처럼 학습하고 판단할 수 있도록 하는 기술입니다.

핵심 개념:
- **머신러닝**: 데이터로부터 패턴을 학습하는 방법
- **딥러닝**: 신경망을 사용한 고급 머신러닝 기법
- **알고리즘**: 문제를 해결하는 단계별 절차
- **모델**: 학습된 AI의 결과물 (예측을 수행하는 프로그램)

1.2 AI 태스크 종류
-----------------

**분류(Classification)**
- 입력을 여러 카테고리 중 하나로 분류
- 예시: 이메일 스팸/정상 분류, 이미지 속 객체 인식
- 출력: 클래스 레이블 (고양이, 개, 새 등)

**회귀(Regression)**
- 연속적인 수치 값을 예측
- 예시: 집값 예측, 주가 예측, 온도 예측
- 출력: 실수 값

**클러스터링(Clustering)**
- 비슷한 특성을 가진 데이터끼리 그룹화
- 예시: 고객 세분화, 유전자 분석
- 출력: 클러스터 그룹

**생성(Generation)**
- 새로운 데이터를 생성
- 예시: 텍스트 생성, 이미지 생성, 음악 작곡
- 출력: 새로운 콘텐츠

1.3 AI 학습 흐름
---------------

**1) 데이터 수집 및 준비**
- 문제에 맞는 데이터 수집
- 데이터 품질 확인 및 정제
- 학습/검증/테스트 데이터 분할

**2) 데이터 전처리**
- 결측치 처리
- 이상치 제거
- 정규화/표준화
- 특성 선택 및 변환

**3) 모델 선택 및 설계**
- 문제 유형에 맞는 알고리즘 선택
- 모델 구조 설계
- 하이퍼파라미터 설정

**4) 모델 훈련**
- 학습 데이터로 모델 학습
- 검증 데이터로 성능 모니터링
- 과적합 방지 및 최적화

**5) 모델 평가**
- 테스트 데이터로 최종 성능 평가
- 다양한 평가 지표 확인
- 오류 분석 및 개선점 도출

**6) 배포 및 운영**
- 실제 환경에 모델 배포
- 성능 모니터링
- 지속적 개선

1.4 AI 모델 설명
---------------

**선형 모델**
- 가장 기본적인 모델
- 입력과 출력 간의 선형 관계 학습
- 예시: 선형 회귀, 로지스틱 회귀
- 장점: 해석하기 쉬움
- 단점: 복잡한 패턴 학습 제한

**의사결정 트리**
- 조건문의 연속으로 결정 과정 모델링
- 규칙 기반 예측
- 장점: 직관적 이해 가능
- 단점: 과적합 위험

**신경망(Neural Networks)**
- 뇌의 뉴런을 모방한 모델
- 여러 층으로 구성된 복잡한 패턴 학습
- 장점: 높은 표현력
- 단점: 해석 어려움, 많은 데이터 필요

**CNN (Convolutional Neural Network)**
- 이미지 처리에 특화된 신경망
- 합성곱 연산으로 이미지 특징 추출
- 이미지 분류, 객체 인식에 주로 사용

**RNN (Recurrent Neural Network)**
- 순차적 데이터 처리에 특화
- 시계열 데이터, 자연어 처리에 사용
- 이전 정보를 기억하며 학습

**앙상블 모델**
- 여러 모델을 결합하여 성능 향상
- 예시: Random Forest, Gradient Boosting
- 개별 모델보다 안정적이고 정확한 예측
