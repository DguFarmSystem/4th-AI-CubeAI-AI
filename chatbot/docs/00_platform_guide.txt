AI 학습 플랫폼 사용 가이드
==========================

이 플랫폼은 AI/ML을 체계적으로 학습할 수 있도록 설계된 교육 플랫폼입니다.

플랫폼 특징
----------
- **4단계 구조화된 커리큘럼**: 새싹 → 잎새 → 가지 → 열매 단계
- **실습 중심 학습**: 이론과 실제 구현을 병행
- **블록 코딩 시스템**: 드래그 앤 드롭으로 AI 파이프라인 구성
- **즉석 피드백**: 실시간 결과 확인 및 성능 분석

4단계 학습 커리큘럼
------------------

**🌱 1단계: 새싹 단계 – AI의 개념 잡기**
- AI 기본 개념 이해
- 다양한 AI 태스크 종류 학습
- AI 프로젝트 전체 흐름 파악
- 기본 모델 유형별 특징 이해

**🍃 2단계: 잎새 단계 – 데이터 전처리 배우기**
- 이미지 데이터 구조 이해
- 전처리의 필요성과 방법 학습
- 실습 블록을 통한 전처리 경험
- 데이터 품질 향상 기법 습득

**🌿 3단계: 가지 단계 – 모델 구조 만들기**
- CNN 핵심 개념 학습
- 레이어별 역할과 순서 이해
- 모델 설계 실습 블록 활용
- 구조 최적화 방법 학습

**🍎 4단계: 열매 단계 – 학습 및 실험**
- 모델 훈련 과정 이해
- 하이퍼파라미터 튜닝 실습
- 성능 시각화 및 분석
- 실험적 개선 방법 학습

블록 코딩 시스템
---------------

**데이터 전처리 블록**
- 데이터 로딩 블록
- 크기 조정 블록
- 정규화 블록
- 데이터 증강 블록
- 데이터 분할 블록

**모델 설계 블록**
- 합성곱층 블록
- 풀링층 블록
- 활성화 함수 블록
- 드롭아웃 블록
- 완전연결층 블록

**학습 및 평가 블록**
- 옵티마이저 설정 블록
- 손실 함수 선택 블록
- 모델 훈련 블록
- 성능 평가 블록
- 결과 시각화 블록

학습 방법론
----------

**순차적 학습**
- 각 단계를 순서대로 완주
- 이전 단계 내용을 충분히 이해한 후 다음 단계 진행
- 실습과 이론을 병행하여 깊이 있는 이해

**반복 학습**
- 어려운 개념은 여러 번 반복 학습
- 다양한 예시와 실습으로 개념 강화
- 실제 프로젝트 적용을 통한 경험 축적

**실험적 학습**
- 다양한 파라미터로 실험해보기
- 실패와 성공을 통한 경험적 학습
- 창의적 아이디어 시도 및 검증

지원 기능
--------

**실시간 도움말**
- 각 단계별 상세 가이드 제공
- 막히는 부분에 대한 즉각적 설명
- 관련 개념 및 배경 지식 제공

**성능 분석**
- 모델 성능 자동 분석
- 개선점 제안
- 비교 분석 도구

**프로젝트 관리**
- 학습 진도 추적
- 완성한 모델 저장 및 관리
- 실험 이력 기록

자주 묻는 질문
-------------

**Q: 프로그래밍 경험이 없어도 괜찮나요?**
A: 네, 블록 코딩 시스템으로 프로그래밍 없이도 AI 모델을 만들 수 있습니다.

**Q: 어느 단계부터 시작해야 하나요?**
A: AI가 처음이라면 새싹 단계부터, 기본 개념을 안다면 자신의 수준에 맞는 단계부터 시작하세요.

**Q: 실습에 필요한 데이터는 어떻게 구하나요?**
A: 플랫폼에서 제공하는 샘플 데이터셋을 사용하거나, 직접 데이터를 업로드할 수 있습니다.

**Q: 모델 성능이 좋지 않을 때는 어떻게 하나요?**
A: 열매 단계의 성능 향상 가이드를 참고하여 체계적으로 개선해보세요.
